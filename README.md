# 项目名称
利用hugin实现移动端图片拼接

## 简介
利用hugin实现移动端图片拼接，基于android平台，把hugin项目移植到安卓平台实现图片拼接成全景图

## 功能特点
素材采集采用鲲图手机全景云台对素材采集，然后把素材导入项目实现移动端完成全景图拼接
